Conectar con GitHub Gist

Pega tu Personal Access Token de GitHub para empezar a sincronizar. Tus datos quedan en un Gist privado de tu cuenta — solo tú accedes a ellos.

Se guarda en este dispositivo. Nadie más que tú lo verá.

Si ya configuraste sync en otro dispositivo, pega aquí el ID del Gist que aparece en esa app. Si es tu primer dispositivo, déjalo vacío y la app creará uno automáticamente.

¿Cómo obtener tu token?

  1. 1. Entra a github.com/settings/tokens y haz click en Generate new token (classic)
  2. 2. Ponle un nombre (ej. "Capital Sync"), elige expiración "No expiration", y marca SOLO la casilla gist
  3. 3. Click en Generate token y copia el código que aparece (empieza con ghp_)
  4. 4. Pégalo arriba y haz click en Conectar. El primer dispositivo creará el Gist automáticamente.
